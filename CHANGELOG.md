## [1.0.0] - 2026‑03‑24

### Added
- Initial release of the CSV Drag‑and‑Drop Plotter.
- Drag‑and‑drop CSV file loading with automatic delimiter/encoding detection.
- Interactive table view with sorting and scrolling.
- Five plot types: scatter, line, bar, histogram, box.
- Customizable axis labels, titles, grid, colors.
- Light/dark theme switching.
- Export plots as PNG, JPEG, PDF, SVG.
- Embedded matplotlib toolbar (zoom, pan, save, reset).
- Comprehensive unit and integration tests (94% coverage).
- User guide, developer guide, API overview, packaging instructions.
